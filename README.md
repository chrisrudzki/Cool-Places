# Cool-Places2

Using webpack to bundle files

using google firebase for local hosting


# to start the project:

firebase --version
(Check if firebase is installed)

firebase login
(should get chrisrudzki7@gmail.com)

npm list webpack

Npm init -y 

npm install --save-dev webpack 

webpack-cli

(initalize webpack)


npm run build
(rebundle files)

npx firebase
firebase serve
(runs local server)
